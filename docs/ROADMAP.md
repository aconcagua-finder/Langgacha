## Фаза 0: Прототип (текущая)

Цель: получить работающий скелет — от базы слов до отображения карточки в браузере.

### Этап 0.1 — Инфраструктура + Seed-база + Генерация карточки ✅
- [x] Docker-окружение (PostgreSQL, Redis, Backend, Frontend)
- [x] Схема БД: таблицы Word (seed-пул), Card (экземпляры карт)
- [x] Seed-генератор: 30 испанских слов с корректными данными
- [x] API: POST /api/cards/generate + GET /api/cards
- [x] React-компонент карточки (лицо + разворот по клику, стили по типам/рарностям)

### Этап 0.2 — Бустеры + Коллекция
- [ ] API бустера: генерация 5 карт с гарантиями рарности
- [ ] UI открытия бустера (анимация раскрытия)
- [ ] UI коллекции: сетка карточек с фильтрами по типу/рарности
- [ ] Привязка карт к пользователю (пока без авторизации — один юзер)

### Этап 0.3 — Боевая система (базовая)
- [ ] 5v5 бой с ботом: выбор колоды → вопросы → автобой
- [ ] Формирование колоды бота
- [ ] Quiz-компонент (4 варианта ответа)
- [ ] Расчёт урона, HP, результат боя
- [ ] Экран результатов и наград

### Этап 0.4 — Прогрессия и экономика
- [ ] Система мастерства (прогресс 0/5 → Dominada)
- [ ] Система ветхости (состояние карточки по времени)
- [ ] Polvo: распыление дубликатов
- [ ] Крафт карточек
- [ ] Прогрессия: разблокировка пулов по освоенным картам
