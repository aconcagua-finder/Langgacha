## Фаза 0: Прототип (текущая)

Цель: получить работающий скелет — от базы слов до отображения карточки в браузере.

### Этап 0.1 — Инфраструктура + Seed-база + Генерация карточки ✅
- [x] Docker-окружение (PostgreSQL, Redis, Backend, Frontend)
- [x] Схема БД: таблицы Word (seed-пул), Card (экземпляры карт)
- [x] Seed-генератор: 30 испанских слов с корректными данными
- [x] API: POST /api/cards/generate + GET /api/cards
- [x] React-компонент карточки (лицо + разворот по клику, стили по типам/рарностям)

### Этап 0.2 — Бустеры + Коллекция ✅
- [x] API бустера: POST /api/boosters/open (5 карт, шансы из GDD, гарантия UC+)
- [x] UI открытия бустера (последовательное появление карт с анимацией)
- [x] UI коллекции: сетка карточек + фильтры по типу/рарности + сортировка
- [x] Модалка просмотра карточки с переворотом
- [x] Роутинг (/, /collection) + навигация TopNav

### Этап 0.3 — Боевая система (базовая) ✅
- [x] 5v5 бой с ботом: выбор колоды → квиз → бой → результаты
- [x] Формирование колоды бота (виртуальные карты, масштабирование ±20%)
- [x] Quiz-компонент (4 варианта ответа, проверка на сервере)
- [x] Расчёт урона/HP, пошаговый combat log, выжившая карта с HP carry-over
- [x] Экран результатов и наград (Polvo, шанс бонусной карты)

### Этап 0.4 — Прогрессия и экономика
- [x] Система мастерства (прогресс 0/5 → Dominada)
- [x] Система ветхости (состояние карточки по времени)
- [x] Polvo: распыление дубликатов
- [ ] Крафт карточек
- [x] Прогрессия: разблокировка пулов по освоенным картам
