# LangGacha — Roadmap разработки

## Статус: Фаза 2 — Контент и полировка (почти завершена)

---

## Фаза 0: Прототип ✅

Цель: работающий скелет — от базы слов до отображения карточки в браузере.

### Этап 0.1 — Инфраструктура + Seed-база + Генерация карточки ✅
- [x] Docker-окружение (PostgreSQL, Redis, Backend, Frontend)
- [x] Схема БД: таблицы Word (seed-пул), Card (экземпляры карт)
- [x] Seed-генератор: 30 испанских слов с корректными данными
- [x] API: POST /api/cards/generate + GET /api/cards
- [x] React-компонент карточки (лицо + разворот по клику, стили по типам/рарностям)

### Этап 0.2 — Бустеры + Коллекция ✅
- [x] API бустера: POST /api/boosters/open (5 карт, шансы из GDD, гарантия UC+)
- [x] UI открытия бустера (последовательное появление карт с анимацией)
- [x] UI коллекции: сетка карточек + фильтры по типу/рарности + сортировка
- [x] Модалка просмотра карточки с переворотом
- [x] Роутинг (/, /collection) + навигация TopNav

### Этап 0.3 — Боевая система (базовая) ✅
- [x] 5v5 бой с ботом: выбор колоды → квиз → бой → результаты
- [x] Формирование колоды бота (виртуальные карты, масштабирование ±20%)
- [x] Quiz-компонент (4 варианта ответа, проверка на сервере)
- [x] Расчёт урона/HP, пошаговый combat log, выжившая карта с HP carry-over
- [x] Экран результатов и наград (Polvo, шанс бонусной карты)

### Этап 0.4 — Прогрессия и экономика ✅
- [x] Система мастерства (прогресс 0/5 → Dominada)
- [x] Система ветхости (состояние карточки по времени)
- [x] Polvo: распыление дубликатов
- [x] Прогрессия: разблокировка пулов по освоенным картам
- [x] Крафт карточек

## Фаза 2: Контент и полировка (текущая)

### Этап 2.0 — Seed-пул и подготовка к мульти-языку ✅
- [x] 150 слов в seed-пуле (75C/45UC/20R/10SR)
- [x] conceptKey для будущего переиспользования ассетов

### Этап 2.1 — UI-полировка карт и коллекции ✅
- [x] Dust → Пыль, гайд обобщён для мульти-языка
- [x] Бустер: клик по карте → полноразмерное превью
- [x] ATK/DEF в нижних углах карты (slate-стиль)
- [x] Три режима коллекции (полноразмерные / мини / таблица)

### Этап 2.2 — Баланс и гарантии ✅
- [x] Формула урона с diminishing returns (ATK × (1 − DEF/(DEF+200)))
- [x] Адаптивный рейд-босс (статы от топ-20 карт игрока)
- [x] Гарантия UC+ в каждом бустере
- [x] Pity SR+ каждые 10 бустеров для всех игроков

### Этап 2.3 — Контент ✅
- [x] Переработка 150 hints (мнемоники вместо грамматических справок)
- [x] AI-генерация 60 картинок по conceptKey (3D miniature стиль)
- [x] Интеграция картинок в CardFace/CardMini с fallback на emoji

### Этап 2.4 — Механики ✅
- [x] Мастерство и ветхость на уровне слова (WordProgress)
- [x] Эволюция карт (освоил слово → +20% статов, сложные вопросы)
- [x] Визуал ветхости (CSS-эффекты: Brilliant/Normal/Worn/Deteriorated)

### Этап 2.5 — Механики ✅
- [x] Новые типы квизов (модуль quiz + reverse тип, mastery-based выбор)
- [x] Бустер: 7 карт, визуал пака, анимации раскрытия, фикс UC+ позиции

### Этап 2.6 — Углубление квизов (в работе)
- [x] Typing quiz — ввод слова вручную, fuzzy-сравнение с диакритикой
- [x] Горячие клавиши для квизов (1-4 для вариантов, автопереход после ответа)
- [ ] TTS озвучка слов (150 слов, статические аудиофайлы)
- [ ] Генерация оставшихся ~77 картинок по conceptKey

### Этап 2.7 — Переработка боя ✅
- [x] Скрытие слова на карте во время квиза (анти-спойлер)
- [x] Квиз для carry-over карты (повторение каждый раунд)
- [x] Карточки в бою → визуал с картинками и рарностью вместо плоских панелей

### Этап 2.8 — Баг-фиксы и полировка боя ✅
- [x] Фиксы визуала: карточки обрезаются, вдохновение не помещается
- [x] Ускорение боёв высоких рангов (минимальный урон от HP)
- [x] Enter/пробел для кнопки "Далее"
- [x] Итоги боя с мини-картами
- [x] Фикс подсветки ответов (correctAnswer с бэкенда)

### Этап 2.9 — Рефакторинг боевой механики ✅
- [x] 5v5 без carry-over: каждая карта = 1 раунд, побед > поражений = победа
- [x] Мини-карты в RoundResult (итог каждого этапа, не только итог боя)
- [x] Автоподбор колоды: приоритет ветхим, дедупликация по слову
- [x] Рандом урона ±20% (rollCombatDamage)

### Этап 2.10 — Полировка боя и автоподбора ✅
- [x] Фикс autoSelect: сильнейший экземпляр из группы, ветхость+рандом между словами
- [x] Компактный боевой экран: карты+квиз на одном экране без скролла
- [x] Визуал итогов: ⚔ вместо галочек, карты ближе
- [x] Бот per-slot scaling (сила бота ±15% от конкретной карты игрока)

- [x] Убрать ручной выбор колоды — только автобой (10 раундов)
