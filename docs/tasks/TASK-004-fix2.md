# TASK-004-fix2: Модалка группы — левая колонка на всю высоту

## Цель

Микрофикс: в модалке стопки (CardGroupModal) левая колонка с экземплярами не занимает всю доступную высоту — внизу остаётся пустое пространство.

---

## Что исправить

В файле `frontend/src/components/collection/CardGroupModal.tsx`:

1. Убрать фиксированный `max-h-[360px]` с грида экземпляров (строка 69). Вместо этого — пусть обе колонки грида растягиваются на всю доступную высоту контейнера.

2. Сделать grid-контейнер (строка 64, `grid grid-cols-1 gap-6 lg:grid-cols-[1fr_360px]`) с `min-h-0 flex-1` чтобы он занимал всё свободное пространство внутри модалки.

3. Левая колонка — `overflow-y-auto` без фиксированной высоты, пусть тянется по высоте грида.

4. Правая колонка (просмотр) — `sticky top-0` чтобы при скролле левой колонки preview оставался на месте.

По сути: обе колонки должны быть одинаковой высоты, заполняя всю модалку. Левая скроллится если карт много, правая — фиксирована.

## Критерии готовности

1. Левая колонка модалки занимает всё пространство до низа — нет пустоты

3. Правая колонка (просмотр) не скроллится вместе с левой
