## 1. Фикс размера карточек в коллекции

### Проблема

В `CollectionGrid.tsx` карточки масштабируются через `transform: scale(0.7)`. Это визуально уменьшает карту, но элемент в layout занимает полный размер (340×480px). Отсюда "дыры" — пустое пространство под и вокруг карточек.

### Решение

Создать компактную версию компонента карточки — `CardMini.tsx`. Это **не масштабированная копия**, а отдельный компонент с уменьшенным layout:

```
frontend/src/components/card/CardMini.tsx
```

Характеристики CardMini:
- Размер: **200×280px** (вместо 340×480)
- Убрать область изображения (она сейчас всё равно заглушка) — вместо неё маленькая иконка типа (эмодзи) в углу
- Слово — крупно, но меньше чем в полной карте
- Перевод — мелко
- FUE / DEF — в одну строку компактно
- Рарность — цветная полоска сверху (вместо бейджа)
- Состояние + мастерство — одна строка внизу
- Рамка карточки — цвет по типу (как в полной версии)

Примерная структура:
```
┌─────────────────────┐
│ ■■■■■ (цвет рарности)│  ← тонкая полоска рарности
│  ⚡                  │  ← иконка типа
│  LABURAR             │  ← слово
│  работать (сленг)    │  ← перевод
│                      │
│  FUE 1850  DEF 1200  │
│  ✨ Normal  ●●●○○    │
└─────────────────────────┘
```

В `CollectionGrid.tsx` заменить CardFace + scale(0.7) на CardMini. Убрать все `transform: scale(...)`.

## 2. Группировка дубликатов

Карты одного и того же слова (из Word) показывать как стопку.

### Логика группировки

В `CollectionPage.tsx` (или вынести в утилиту) группировать карты по `word` (строке):

```typescript
type CardGroup = {
  word: string;
  cards: GeneratedCard[];
  bestCard: GeneratedCard; // с наибольшим FUE+DEF
};
```

### Визуал стопки

В `CollectionGrid.tsx`:
- Если группа содержит 1 карту — обычная CardMini
- Если группа содержит 2+ карт:
  - Показать CardMini верхней (лучшей) карты
  - За ней видны "тени" остальных карт (сдвинутые на 4px вправо и вниз каждая, максимум 3 тени)
  - В правом верхнем углу — бейдж с количеством: "×3"
  - По клику — открывается модалка со списком всех экземпляров этого слова

### Модалка группы

Обновить `CardModal.tsx` или создать `CardGroupModal.tsx`:
- Если кликнули на стопку — показать список всех экземпляров:
  - Горизонтальный скролл полноразмерных карточек
  - Или вертикальный список с компактным отображением (CardMini)
  - Каждая карточка кликабельна → показывает полноразмерную карту с переворотом
- Под каждой картой кнопка "Распылить" (пока неактивная, визуально есть но disabled с тултипом "Скоро")
