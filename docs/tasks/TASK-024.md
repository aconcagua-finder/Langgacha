## 1. Коллекция — показывать полноразмерные карты (CardFace), не CardMini

Импортировать CardFace вместо CardMini.

Карточки должны быть в сетке — адаптировать grid:

```

```

Максимальная ширина каждой ячейки — не больше 300px, чтобы карты не были огромными на десктопе. Можно через `max-w-[300px] mx-auto`.

## 2. Вернуть визуальные стопки для дубликатов

Для карт с count >= 2 — показывать "тени" позади карты, создающие эффект стопки.

### Правильная реализация (адаптивная):

Не использовать захардкоженные пиксели! Тени должны масштабироваться с картой:

```tsx
{isStack ? (
  <>
    {Array.from({ length: Math.min(3, count - 1) }, (_, i) => (
      <div
        key={i}
        className="absolute inset-0 rounded-2xl border border-slate-800/60 bg-slate-900/30"
        style={{
          transform: `translate(${(i + 1) * 6}px, ${(i + 1) * 6}px)`,
          zIndex: -(i + 1),
        }}
      />
    ))}
    <div className="absolute -right-1 -top-1 z-20 rounded-full bg-slate-950/80 px-2 py-1 text-xs font-extrabold text-slate-50 shadow">
      ×{count}
    </div>
  </>
) : null}
```

Ключевое: тени используют `inset-0` (заполняют весь контейнер родителя), а не фиксированные width/height. Сдвиг через translate — это нормально, 6px — мелкий offset.

Контейнер карты должен быть `relative` чтобы тени позиционировались правильно.
